# Correio San Andreas
## Teste do processo seletivo Take.

A aplicação é uma solução em *C#* para realizar a busca pela rota mais rápida para a entrega de encomendas do correios em San Andreas.
Nela é realizada a leitura de um arquivo no formato _TXT_ com um padrão de dados de entrada para os trechos e encomendas diárias.
O retorno da aplicação é um novo arquivo _TXT_ com os melhores resultados.

Projeto criado utilizando:
 * Visual Studio 2019 versão 16.5.5 
 * .NET Core
 * Windows 7
 * XUnit
